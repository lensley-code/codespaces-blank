# Instagram Showcase Feature

This is a standalone website feature intended to be built and shipped in parallel to other ongoing work.

## What it includes

- Responsive horizontal showcase of social-style cards
- Play-button overlay and episode badges
- CTA row with `Load More` and `Follow on Instagram`
- Isolated HTML, CSS, and JavaScript files for easy integration

## Files

- `index.html` — demo implementation and structure
- `styles.css` — complete feature styling
- `feature.js` — lightweight interaction logic

## Integrate into your site

1. Copy the `<section class="instagram-showcase">...</section>` block from `index.html`.
2. Include `styles.css` in your main page head.
3. Include `feature.js` before the closing body tag.
4. Replace image URLs and labels with your own content feed.

## Notes

- The feature is fully self-contained and does not require a framework.
- You can swap the static list for API data later without changing the CSS layout.
